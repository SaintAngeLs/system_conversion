==========================Gdzie znajduje się projekt?===================

Projekt znajduje się za ściezką: _calculator_project_\Source\_calculator_project_\calculator\calculator_

Dla uruchomienia projektu, trzeba uruchomić decyzję calculator_.sln w Visual Studio 2017.

Dla uruchomienia programu, trzeba uruchomić plik calculator.exe.

==========================Jak korzystać z programu?=====================

1. Po uruchomieniu program wyświerla komunikat :

"Please, press 1 if you want to work with integer numbers (less than 1*10^4).

Please, press 2 if you want to work with big integer numbers (less and greater then 1*10^4)."

, w którym trszeba naćinąć 1, jeśli użytkowik chce wykonywać operacje na liczbach całkowitych mniejszych 10000, lub 2, jeśli użytkowik chce wykonywać operacje na dużych liczbach całkowitych.


2. Pojawi się komunikat: 

"Please, press 1 if you want to obtain the results of every operation in this file.

Please, press 2 if you want to obtain the result of operation in one separate file
(program exits after obtaining the result of operation)."

, w w którym trszeba naćinąć 1, jeśli użytkowik chce otrzymać wynik każdej operacji w jednym pliku, lub 2, jeśli użytkowik chce otrzymać wynik każdej operacji w odzielnym pliku (program zamyka siępo wykonaniu operacji).

3. Program wyświetla komunikat:

"Please, input the sign of operation, space and the base of the system to execute operation.
Press 'ENTER'. After that, input the first argument, press 'ENTER' and then input the second agrument, press 'ENTER'."

, który mówi że format danych wejściowych jest taki:

	operacja system_liczbowy

	argument_1

	argument_2

	wynik

, gdzie ( ) to szpacja;

Przykłady są dostępne w:
 ..\_calculator_project_\Input\input_1

 ..\_calculator_project_\Input\input_2  //dla dużych liczb

 ..\_calculator_project_\Input\output_1

 ..\_calculator_project_\Input\output_2 //dla dużych liczb
